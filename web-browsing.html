Web Browsing

Browser

An application software to access the world wide web. The browser must retrieve the neccessary content from a web server and display the page.

Web Server

A web server is computer software and underlying hardware that accepts requests via HTTP, , the network protocol created to distribute web pages, or its secure variant HTTPS.

HTTP

HTTP (Hypertext Transfer Protocol) is an application layer protocol for connecting you to web servers on the internet.

HTTPS, SSL, TLS

HTTPS and TLS are used for secure communication over a computer network, and is widely used on the internet. HTTPS is encrypted using TLS (Transport Layer Security) which was designed to provide security over a network. TLS was formerly SSL (Secure Sockets Layer).

GET and POST

Both requests both interact with a server/database, but have different purposes. The GET request is to fetch information from a server, while the POST request edits specific information inside a database.

HTML

HTML is the language that defines the meaning and structure of web content. HTML is used in most websites, along with CSS and Javascript to add style and functionality of a website.
